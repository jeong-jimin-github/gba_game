# GBA ゲーム

## 説明
C言語学習のためのゲームボーイ・アドバンス(GBA) 向けのゲーム開発プロジェクトです。
敵の弾幕を避け、攻撃してスコアを上げることが全部のシンプルな横スクロールゲームです
